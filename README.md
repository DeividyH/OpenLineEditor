# OpenLineEditor open-source project 
Urban lines editor, based on openstretmap, has features for application development that consume data
in the GTFS General Transit Feed Specification standard. This project exports data in the GTFS standard, in addition, the user can
register new lines and public transport agencies.

## This is project use
+ [Laravel 5.3](https://laravel.com/docs/5.3)
+ PHP >= 7.0.0
+ [Materialize](http://materializecss.com/)
+ [jQuery](https://jquery.com/)
+ [Leaflet](http://leafletjs.com/)
